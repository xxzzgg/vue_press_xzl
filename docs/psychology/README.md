# 心理学专题

这个专题整理一些实用、可落地的心理学主题，不追求学术术语的堆砌，而是关注一个更直接的问题：

**人为什么会这样想、这样感受、这样做，以及我们能怎么调整。**

## 目录

### [认知偏差：大脑常见的判断误区](./01-cognitive-biases.md)

- 为什么人会高估自己已有观点
- 为什么我们总是更容易记住“证明自己是对的”信息
- 为什么损失带来的痛苦比收益更强烈

### [情绪调节：情绪不是敌人，而是信号](./02-emotion-regulation.md)

- 焦虑、愤怒、羞耻感的常见来源
- 情绪上来时，怎样先稳住自己
- 如何区分“真实问题”和“情绪放大”

### [边界感与关系：亲密不等于没有边界](./03-boundaries-and-relationships.md)

- 什么是健康边界
- 为什么很多关系问题本质上是边界混乱
- 如何表达需求而不是压抑或爆发

### [自我成长：从自我攻击到稳定改变](./04-self-growth.md)

- 为什么很多人知道道理却做不到
- 习惯改变为什么总在情绪压力下失败
- 如何建立更现实、更稳定的成长路径

### [交易中的等待与无聊：执行力的真正考验](./05-trading-waiting-and-boredom.md)

- 为什么“手痒”比不会分析更危险
- 等待 setup 时，大脑在经历什么
- 如何把无聊变成纪律，而不是冲动交易

### [亏损、追损与复仇交易：最容易毁账户的心理链条](./06-trading-loss-and-revenge.md)

- 为什么一笔亏损后人会立刻想扳回来
- 追损背后不是勇敢，而是情绪失控
- 如何在亏损后切断错误循环

### [连续亏损后的恢复：先找回节奏，不要急着回本](./07-trading-drawdown-recovery.md)

- 为什么回撤期最危险的不是市场，而是自己
- 连亏之后应该缩小什么、保留什么
- 如何从“想回本”切换到“先恢复执行”

## 阅读建议

1. 如果你最近经常内耗，先看“认知偏差”和“情绪调节”。
2. 如果你在人际关系里容易委屈、愤怒或反复拉扯，先看“边界感与关系”。
3. 如果你总是立目标又放弃，再看“自我成长”。
4. 如果你在交易里总是冲动下单、亏后上头、连亏后乱做，优先看三篇“交易心理学”。

## 一句话核心

> 很多问题不是你不够努力，而是你没有先看清自己的心理机制。
