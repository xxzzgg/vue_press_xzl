# 情绪调节：情绪不是敌人，而是信号

很多人一有负面情绪，就会立刻想把它“压下去”。

但情绪本身不是问题，真正的问题往往是：

- 你不知道它在提醒什么；
- 你一上头，就让情绪替你做决定。

## 1. 常见情绪背后的心理信号

### 焦虑

- 焦虑常常来自不确定感和失控感。
- 你不是单纯“想太多”，而是在试图提前规避风险。

### 愤怒

- 愤怒常常意味着边界被侵犯、期待落空，或者你感到自己不被尊重。
- 很多时候，愤怒底下其实是委屈、无力和害怕。

### 羞耻感

- 羞耻感不是“我做错了”，而是“我这个人有问题”。
- 它会让人回避、讨好、假装没事，或者过度自我攻击。

## 2. 情绪上来的时候，先不要解决人生

情绪高峰期不适合做重大决定。

先把目标改成一件更现实的事：

**先稳定，再判断。**

## 3. 一个简单的调节步骤

### 第一步：命名情绪

- 不要只说“我很烦”。
- 尝试说得更具体：`我现在是焦虑、委屈、愤怒，还是羞耻？`

命名会让情绪从一团混乱，变成可以观察的对象。

### 第二步：回到身体

- 放慢呼吸。
- 离开刺激现场几分钟。
- 喝水、走动、洗脸，让身体先降下来。

情绪失控时，先调身体，比先讲道理更有效。

### 第三步：区分事实和脑补

- 事实：对方说了什么、做了什么。
- 脑补：`他就是看不起我`、`我肯定完了`、`事情一定会更糟`。

很多痛苦并不完全来自事实，而来自自动化解释。

## 4. 不要把调节等同于压抑

- 调节不是假装没情绪。
- 调节也不是“成熟的人就不该难过”。
- 调节的意思是：你允许情绪存在，但不让它直接掌控行为。

## 5. 一个可执行的日常练习

每天花 3 分钟记录三件事：

1. 今天我最强烈的情绪是什么。
2. 它是被什么触发的。
3. 我真正需要的是什么。

长期做这件事，你会逐渐看见自己最常见的触发点和反应模式。
