# 自我成长：从自我攻击到稳定改变

很多人以为成长靠的是“更狠一点”“更自律一点”“再逼自己一下”。

短期也许有效，但长期往往会变成：

- 目标越来越高；
- 对自己越来越失望；
- 最后靠逃避来缓解压力。

## 1. 为什么知道很多道理，还是做不到

因为改变不只靠认知，还受三件事影响：

- 情绪承受能力；
- 环境设计；
- 旧习惯带来的即时奖励。

所以“知道”和“做到”之间，差的往往不是智商，而是系统。

## 2. 自我攻击为什么会让人更难改变

- 自我攻击会迅速拉高压力。
- 压力一高，人就更想回到熟悉、快速、能止痛的旧模式。
- 于是你越骂自己，越可能回到原来的行为。

这也是为什么很多人一边讨厌自己，一边重复同样的问题。

## 3. 更有效的改变方式

### 从小处开始

- 不要把目标设成“我以后彻底变了”。
- 改成足够小、可以持续的动作。

例如：

- 不要说“每天读书两小时”，改成“每天读 10 分钟”。
- 不要说“以后绝不情绪化”，改成“情绪上来时先停 3 分钟再回消息”。

### 让行动容易发生

- 想坚持写作，就把文档提前打开。
- 想少玩手机，就把最容易沉迷的应用移出首页。
- 想规律作息，就先固定起床时间，而不是同时要求早睡早起。

### 用复盘代替羞辱

失败后不要立刻问：`我怎么又这么差？`

改问三个问题：

1. 我是在什么情境下失败的？
2. 当时我最真实的需求是什么？
3. 下次我可以把门槛再降低到什么程度？

## 4. 稳定成长的核心

真正能持续的成长，不是靠一阵子的热血，而是靠：

- 对自己的模式越来越清楚；
- 对失败没那么羞耻；
- 每次只改一点，但一直在改。

成长不是一场自我审判，而是一场长期校准。
