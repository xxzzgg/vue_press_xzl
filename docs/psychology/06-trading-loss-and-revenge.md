# 亏损、追损与复仇交易：最容易毁账户的心理链条

一笔亏损真正可怕的地方，往往不在于亏掉的钱，而在于它接下来会诱发什么行为。

很多账户不是死于一次止损，而是死于：

**止损之后，情绪接管了系统。**

## 1. 为什么亏损后人会立刻想扳回来

因为亏损带来的不只是金钱损失，还有三种心理冲击：

- 我是不是看错了；
- 我是不是不行；
- 我必须马上证明刚才只是意外。

于是“下一笔交易”的目的，悄悄从执行系统变成了修复自尊。

这就是复仇交易的开始。

## 2. 复仇交易的典型过程

### 第一步：正常亏损

- 原本只是系统内的一笔止损。
- 如果能按计划处理，它本来完全可控。

### 第二步：情绪不接受

- 心里开始出现“这单不该亏”“市场在针对我”“我得赚回来”。
- 此时你已经不是在看市场，而是在和结果较劲。

### 第三步：标准放宽

- 原本不够好的 setup，也开始被你解释成“可以做”。
- 甚至会提早进场、加大仓位、忽略止损。

### 第四步：连环错误

- 亏损扩大，情绪更重。
- 情绪更重，决策更差。
- 最后你不是在交易，而是在追着亏损跑。

## 3. 为什么追损这么上瘾

因为它会短暂给你一种强烈的主观感觉：

- 我还没输；
- 我还能掌控；
- 只要下一笔回来，一切就恢复正常。

问题在于，这种感觉通常只会把你带向更差的位置。

追损不是勇敢，而是无法接受不确定性和失败感。

## 4. 亏损后最该做的不是分析市场，而是先降温

亏损后，大脑很容易进入“缩小世界”的状态，只盯着一件事：

`把刚才亏掉的赚回来。`

这时你最需要做的不是马上找下一单，而是先切断这个反应链。

### 一个简单规则

只要你脑中出现下面这些句子，就暂停：

- `我必须马上赚回来`
- `这次我加大一点就回来了`
- `刚才只是运气差`
- `这笔我一定不能再错过`

这些都不是分析，而是情绪信号。

## 5. 亏损后的处理流程

### 1. 先停

- 离开屏幕几分钟。
- 不要立刻找下一笔。
- 先让身体和注意力降下来。

### 2. 再分辨

- 这是系统内正常亏损，还是执行错误？
- 如果是系统内亏损，就接受它。
- 如果是执行错误，就记录错误类型，不要急着靠下一笔修复。

### 3. 最后决定是否继续

只有当你能平静回答下面三个问题，才考虑继续交易：

1. 我现在不是为了回本而下单。
2. 我接下来做的 setup 和刚才无关。
3. 即使再亏一笔，我也不会破坏今天的风控。

## 6. 一个比“回本”更重要的目标

真正成熟的交易者，在亏损后最优先保护的不是收益，而是：

- 节奏；
- 判断质量；
- 系统完整性。

因为单笔亏损并不致命，情绪化地连续犯错才致命。

## 7. 一句话原则

> 亏损后第一件事不是证明自己，而是防止自己变形。
