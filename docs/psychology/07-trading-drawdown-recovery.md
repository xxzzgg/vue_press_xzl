# 连续亏损后的恢复：先找回节奏，不要急着回本

连续亏损最危险的地方，不只是账户缩水，而是你会越来越难保持原来的自己。

很多人一进入回撤期，脑子里只剩一个念头：

**快点回本。**

但这恰恰会让回撤变得更深。

## 1. 为什么连亏之后最容易失控

因为连续亏损会同时打击三样东西：

- 对系统的信任；
- 对自己的信任；
- 对未来结果的耐心。

这时人会自然想通过“更快、更大、更频繁”的方式扳回来。

于是常见动作是：

- 提高仓位；
- 放宽 setup；
- 增加交易频率；
- 在本来该休息的时候硬做。

这些动作看起来是在积极解决问题，实际上是在扩大问题。

## 2. 回撤期最重要的目标是什么

不是立刻回本，而是先恢复稳定执行。

因为如果执行已经变形，你就算短期靠运气赚回来，也很快会再吐回去。

回撤期真正该问的是：

- 我是不是还在做同一套东西；
- 我的风险是不是还在控制内；
- 我的错误是市场带来的，还是我自己放大的。

## 3. 恢复期该怎么做

### 1. 缩小仓位

- 回撤期先把仓位降到你几乎感觉不到压力的水平。
- 目的不是赚钱，而是恢复正常判断。

### 2. 缩小交易范围

- 只做你最熟、历史数据最稳定的 setup。
- 不要在状态差的时候尝试新东西。

### 3. 缩小目标

- 不要盯着“把这周亏的都赚回来”。
- 改成更现实的目标，例如：
  - 今天只做系统内的单；
  - 今天不犯追单错误；
  - 今天能做到按计划止损。

## 4. 如何判断问题到底出在哪

把最近的亏损分成两类：

### A 类：系统内亏损

- setup 合规；
- 风控合规；
- 执行合规；
- 只是结果不好。

这类亏损不说明你坏了，它只是交易的一部分。

### B 类：执行性亏损

- 乱追；
- 提前做；
- 不按止损走；
- 情绪化加仓；
- 为了回本而下单。

这类亏损才是恢复期最需要优先处理的。

## 5. 一个非常实用的恢复原则

在连续亏损阶段，先不要想着“怎么赚更多”，先做到：

- 不放大错误；
- 不增加暴露；
- 不把普通回撤变成失控崩盘。

能守住这一点，你就已经比大多数人强很多。

## 6. 什么时候算恢复了

不是账户一回本就算恢复。

真正的恢复标志更像是：

- 你又能等待；
- 你又能接受空仓；
- 你又能看见 setup，而不是只看见盈亏；
- 亏损时你还能按规则处理，而不是情绪化乱动。

当这些状态回来，收益只是时间问题。

## 7. 一句话总结

> 回撤期先修执行，再谈盈利；先活下来，再谈回本。
